# Diccionario de Datos

## asambleista
- asambleista_id: Identificador único
- cedula: Identificación oficial
- nombre: Nombre completo
- correo_institucional: Correo oficial

## nombramiento
- fecha_inicio / fecha_fin: Control histórico del cargo
- sector_id: Sector asociado AL PERIODO, no a la persona

## elemento_normativo
- id_elemento_padre: Permite jerarquía recursiva
- fecha_inicio_vigencia / fecha_fin_vigencia: Control de versiones normativas
